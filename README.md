# simple-api-server

A simple API server in Python to respond to challenge requests. Essentially boilerplate, but it will respond to JSON requests with

```json
{
    "challenge": "35902439"
}
```

and respond to everything else with the fields replaced with `data`.

## Usage

To run, run

```bash
python app.py
```
